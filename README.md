# structure-pug-sass-webpack

# Structure

src
    assets
        images
        fonts
    inline-script
    lib ( store 3rd )
    scripts ( script )
        bootstrap (file js to build )
    styles
    template
        components
        pages

# Path URL assets
template : "./assets"

## Installation

Run `yarn install` (Install packages)

## Build

Run `yarn start` (build develop)
Run `yarn build` (build prod)
